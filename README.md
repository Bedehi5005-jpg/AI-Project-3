# 🎯 AI Recommendation System

A simple **Rule-Based AI Recommendation System** built with Python. This beginner-friendly project recommends **Movies, Books, and Music** based on the user's selected category and favorite genre.

---

## 📌 Features

- 🎬 Movie recommendations
- 📚 Book recommendations
- 🎵 Music recommendations
- Beginner-friendly Python code
- Uses conditional statements (`if`, `elif`, `else`)
- Interactive command-line interface

---

## 🛠️ Technologies Used

- Python 3
- VS Code (Recommended)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Recommendation-System.git
```

2. Open the project folder

```bash
cd AI-Recommendation-System
```

3. Run the Python file

```bash
python ai_recommendation.py
```
